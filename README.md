# Product Grid Card

A customizable product card widget for Flutter.

## Usage

```dart
import 'package:product_grid_card/product_grid_card.dart';

ProductGridCardV2(
  image: 'https://example.com/product.jpg',
  title: 'Product Name',
  subtitle: 'Short description',
  price: 100.0,
  discountPercent: 20,
  productId: 'abc123',
);
```
